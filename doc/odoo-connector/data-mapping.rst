====================
Data mapping details
====================

This image shows a high-level overview of the data mapping between the Odoo and frePPLe.
Disclaimer: it's a simplified and slightly outdated snapshot.

The connector doesn't cover all possible configurations of Odoo and frePPLe.
The connector will very likely require some customization to fit your particular
Odoo configuration and the planning requirements in frePPLe.  You do this by
forking the `github repository https://github.com/frePPLe/odoo <https://github.com/frePPLe/odoo>`_.
